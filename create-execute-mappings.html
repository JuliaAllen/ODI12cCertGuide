Oracle Data Integrator (ODI) 12c

  • Create and execute an ODI mapping
Order to create mappings:
  1. Create a New Mapping
  2. Adding and removing components
  3. Connecting and configuring components
  4. Defining a physical configuration
  5. Run mapping
Open the logical diagram on the mapping editor and begin to build your mapping.
  1. Design Navigator > Mappings node (under the project you want to create the mapping for) and right click New Mapping
  2. Fill in the name, description, etc
  3. Drag the components you need on to the staging area
  4. Drag datastores/reusable mappings from the Design Navigator
  5. Connect the components to create the data flow
  6. Define the expressions and conditions by selecting the attribute you wish to define
  7. For expressions click on the Target tab, for conditions click on the condition tab
  8. Define the physical connection by going to the physical tab > define the loading and integration strategies
  9. Run the mapping by right-clicking on the mapping from the Design Navigator and choosing Run
