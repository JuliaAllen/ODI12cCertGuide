Oracle Data Integrator (ODI) 12c

  • Create a Datastore - Manual Way
If you have a blank model and a flat file, here is how you can manually create and populate a datastore.

  • Go to the Designer Navigator and choose a model or sub-model
  • Right click and choose New Datastore
  • On the definition tab, name the datastore, resource name and alias
  • For flat files: on the File tab go to file format and choose the type, fill in if there is a header, record separator and field separator define the characters used to separate the lines and then click save
    •

    Now the datastore is created.

    Reverse-Enginnering Fixed File Datastores

    1. Go to the Attributes tab that has a fixed format
    2. Click the Reverse-Engineer button
    3. Click on the riler to create marketing delimited the attributes
    4. Attributes are created with pre-generated names - edit by clicking in the header line
    5. You must set the Attribute Name, Datatype and Length for each attribute (no spaces) and click ok


    Reverse Engineer Delimited Files:
    1. Go to the Attributes tab
    2. Click reverse-engineer
    3. Edit the attributes according to your content and save
