Oracle Data Integrator (ODI) 12c

  • Describe the use of reusable mappings
Reusable mappings allow you to take multi-step integrations and make them into a single component that you can use in other mappings. It is a projector component. To create a reusable mapping:
  1. Designer Navigator > click on a project > right-click reusable mappings > new reusable mapping
  2. enter the name and drag datastores, components and assemble mapping logic
  3. Validate your mapping and click save
